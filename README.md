## Expense Tracker Dashboard

A full-stack MERN application built to track personal finances. It features a dynamic dashboard with data visualization, real-time summaries, and advanced time-period filtering.

![Dashboard Preview](assets/dashboard.png)

Key Features

Interactive Dashboard: Visual bar charts (Chart.js) to track spending trends.
Smart Filtering: Filter data by "Last 7 Days", "Last 30 Days", or specific "Month & Year".
Category Breakdown: Auto-calculated summaries of top spending categories.
Transaction History: Add, view, and delete expense records instantly.
Responsive UI: Clean, modern interface built with React and CSS Grid.

Tech Stack

Frontend: React (Vite), Chart.js, Axios
Backend:Node.js, Express.js, REST API
Database: MongoDB
